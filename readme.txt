app.js:

- Necessário mudar 'username' e 'password' na string de conexão com o mongoDB no arquivo 'app.js' por um username e password válido.
- Caso queira mudar o nome do banco de dados, altere a palavra 'aprendendo' na string de conexão com o BD pelo de sua preferência.

server.js:

- A porta da aplicação está configurada para 3003, caso deseja mudar vá no arquivo 'server.js' e altere para a porta que preferir.
